# /status 命令

## 用法

```
/status                   # 显示当前状态
/status S-xxx             # 显示指定 Story 状态
```

## 说明

显示项目当前状态和进度概览。

## 执行流程

```
1. 读取 STATE.yaml
2. 读取 Stories 和 Tickets
3. 计算进度
4. 输出状态报告
```
