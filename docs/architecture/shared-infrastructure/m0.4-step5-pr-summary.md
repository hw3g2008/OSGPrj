## 1. 概述

将三端（Mentor / Assistant / Lead-Mentor）`MainLayout.vue` 中重复的 sidebar 实现抽取为共享组件 `@osg/shared/components/AppSidebar`，三端 wrapper 减重 **~51.6%**，开启 Assistant-as-SSOT 路线图 M0 阶段的实质化进展。

## 2. 战略背景

- **顶层路线图**：[Assistant-as-SSOT 路线图 v1.1](../2026-04-24-assistant-as-ssot-roadmap.md)
- **共性分析**：整壳 MainLayout 共性 18.7%（严格）/ 66.8%（放宽）< 50% 阈值 → 改抽 AppSidebar 子组件（共性 ~95%），见 [m0.4-mainlayout-commonality.md](./m0.4-mainlayout-commonality.md)
- **抽取方案（已通过 /validate-doc 多轮思辨）**：[m0.4-step3-app-sidebar-extraction-plan.md](./m0.4-step3-app-sidebar-extraction-plan.md)

## 3. 量化收益

### 3.1 代码减重

| 端 | 改造前 | 改造后 | 净减 | 减幅 |
|---|---|---|---|---|
| Mentor | 506 | 240 | -266 | -52.6% |
| Assistant | ~530 | 261 | -269 | -50.7% |
| Lead-Mentor | 527 | 256 | -271 | -51.4% |
| **三端合计** | **~1,563** | **757** | **-806** | **-51.6%** |

含共享代码后总量：1 份共享 367 行 + 3 份 wrapper 757 行 = **1,124 行**（vs 改造前 1,563 行 → -28%）。

**更重要的是**：未来 sidebar 任何变更只改 1 处（`shared/components/AppSidebar.vue`），三端自动同步。
