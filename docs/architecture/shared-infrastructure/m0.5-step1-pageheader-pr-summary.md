### 6.2 视觉对比标准

- **D-Bilingual P1（默认）**：标题中文 22px / 600 + 英文 14px / 500 / `#94a3b8`；description 14px / 500 / `#64748b`。
- **D-Bilingual P1-Compat（兜底）**：若用户主观觉得"标题太大或太小"，切换到此变体（21px / 13px）— 在 `shared/PageHeader/index.scss` 中调整即可。
- **回归基线**：以 PR 合入前的 5 端原型截图（`docs/architecture/shared-infrastructure/m0.5-pageheader-baseline-screenshots/`）作为视觉 diff 参照。

- **必须**：8 个关键 view 视觉与原型 1:1（标题层级、字号、颜色、间距、actions 位置）。
