# 岗位信息页桌面端比例回收设计

日期：2026-03-20

## 背景

`/career/positions` 已按原型重建结构，但当前桌面端视觉比例整体偏大。主要问题集中在标题区、统计卡、筛选栏、下钻条和底部摘要的字号、间距、圆角与阴影强度，导致页面较原型更厚、更胀。

本轮只做桌面端比例回收，不调整交互、数据逻辑、路由、接口或组件结构。

## 目标

- 将页面桌面端视觉密度收回到原型节奏
- 保持当前已接通的真实功能和自动化选择器不变
- 保留现有响应式断点逻辑，避免影响窄屏行为

## 范围

仅调整：

- 页面标题区
- 视图切换和顶部操作按钮
- 统计卡
- 筛选工具栏
- 下钻行业条、公司条、展开表格容器
- 底部摘要和流程图例

不调整：

- 任何接口调用和数据映射
- DOM 结构和交互流程
- 弹窗结构
- 列表/下钻的业务逻辑

## 方案

### Header

- 主标题从当前偏大的字号收回到接近原型的 `36px`
- 英文副标题和副文案的间距同步收紧
- 视图切换容器收回到更小的 padding 和 radius
- 顶部操作按钮按 prototype `btn-sm` 密度回收高度、左右 padding 和圆角
- 浏览统计提示弱化，回到原型的轻提示层级

### 统计卡

- 卡片内边距从厚卡改为短卡
- 数字字号回收到 `28px`
- 标签字号回收到 `12px`
- 卡片圆角与阴影同时减弱，恢复 prototype 的轻量卡片感
- 网格间距回收到 `12px`

### 筛选栏

- 外层容器 padding、圆角、阴影减弱
- 所有控件高度统一回收到 `32px`
- 各下拉宽度按原型收回到 `90-110px` 区间
- 搜索框宽度收回到 `160px`
- 搜索右侧引导按钮缩小，不再与主操作按钮竞争视觉

### 下钻区

- 行业条外层壳体圆角和阴影减弱
- 行业条 padding 收回到 `12px 16px`
- 行业名称、badge、pill、学员数字全部下调一档
- 公司行容器、logo、官网按钮、展开表格 padding 同步下调
- 保持配色与层级不变，只回收尺度

### 摘要与图例

- 摘要区字体回收到 `12-13px`
- 图例容器改回原型的细长浅色条
- 页面底部留白收紧，避免整体放大感

## 验证

- 新增样式回归测试，锁定关键字号和容器尺寸
- 运行 `pnpm exec vitest run src/__tests__/positions.spec.ts`
- 运行 `pnpm build`
- 用浏览器真实打开 `/career/positions` 做桌面端视觉回查
